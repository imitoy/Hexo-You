# Hexo You Theme

A clean and minimal Hexo theme for your blog, inspired by [Material You](https://m3.material.io/), [Materialize](https://github.com/Can1425/Hexo-Theme-Materialis) and based on [MDUI](https://mdui.org/).

**This project is under developing.**

## Features

- Responsive design
- Dark/Light mode support
- Fast performance
- SEO optimized
- Code syntax highlighting
- Mobile-friendly

## Installation

```bash
git clone https://github.com/imitoy/Hexo-You.git themes/hexo-you
```

## Configuration

Update your `_config.yml`:

```yaml
theme: hexo-you
```

## Usage

Place your blog posts in the `source/_posts` directory as Markdown files.
